📘 My HTML Learning Journey

By: Khush Patel | Year: 2025 | Institution: RRC Polytech

✨ Introduction

Over the past few weeks, I practiced and built several mini projects using only HTML. This journey helped me understand how websites are built from scratch and how every tag plays an important role in structure and meaning.

🧱 Projects I Built

  • Personal Bio Page: Practiced headings, paragraphs, and image embedding.
  • Swadhyay Schedule: Used lists to create a simple weekly event schedule.
  • Basic Resume: Structured my education and skills using semantic tags.
  • Fastfood Menu: Learned to use tables for displaying structured data.
  • Feedback Form: Created a working form using inputs, labels, and dropdowns.
  • Portfolio Home Page: Linked all my HTML projects neatly for easy navigation.
  • DF YouTube Project: Built a “Distraction-Free YouTube” with embedded Coding, Music, and Gaming videos — a focused space to learn and relax.

💡 Key Concepts I Learned

  • HTML structure: <!DOCTYPE html>, <head>, <body>
  • Text formatting using <b>, <i>, <strong>, and <em>
  • Navigation with <a> and target="_blank"
  • Lists (<ul>, <ol>) and Tables (<table>, <tr>, <td>)
  • Forms and user inputs with <input>, <textarea>, and <label>
  • Embedding media using <img> and <iframe>
  • Creating links between multiple pages for real project navigation
  • Semantic tags for cleaner, more meaningful code

🧠 Reflection

In the beginning, I only saw HTML as a basic markup language. But after creating these projects, I realized how it forms the foundation of every website. Even without CSS or JavaScript, HTML alone can bring structure, purpose, and clarity.

The DF YouTube Project especially reminded me that focus matters — simplicity and purpose can turn even small projects into something valuable. This practice also strengthened my discipline and consistency.

🚀 What’s Next

My next goal is to learn CSS for styling and JavaScript for interactivity. I’ll keep using GitHub to track my progress and showcase my work professionally.

My mantra: “Don’t aim for perfection — aim for consistent progress.”

HTML was my first step in becoming a Full Stack Developer — and I’ll continue walking this path with focus, awareness, and faith.
