Hey there! I’m **Khush Patel**, a student at **RRC Polytech**, learning **Application Development & Delivery**. Over the past week, I challenged myself to build **6 mini HTML projects** to get stronger at web development fundamentals. This is my journey so far. 😎

## My Projects 🛠️

1. **Personal Bio Page**  
   Practiced headings, paragraphs, adding images, and organizing content in a clean way.  

2. **Swadhyay Schedule**  
   Used lists (`ul` & `li`) to show weekly events clearly.  

3. **Resume**  
   Structured my skills and education into sections using headings, lists, and paragraphs.  

4. **Fastfood Menu**  
   Made a table to show dishes, descriptions, and prices — practiced layout with tables.  

5. **Feedback Form**  
   Created a form with text inputs, textarea, dropdown select, and submit button — got hands-on with forms.  

6. **Portfolio Home Page**  
   Linked all my projects together, practiced navigation, and organized pages nicely.  

## Key Things I Learned 💡

- How to structure a webpage: `DOCTYPE`, `html`, `head`, `body`  
- Text formatting: `h1–h3`, `p`, `strong`, `em`  
- Links & navigation: internal & external `a` tags  
- Lists: `ul`, `li`  
- Tables & forms: `table`, `input`, `textarea`, `select`  
- Adding images: `img`  
- Importance of **semantic HTML** for accessibility and clear structure
